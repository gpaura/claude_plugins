# gpaura Claude Code Plugin

Personal Claude Code setup with productivity skills and specialized AI agents for modern web development.

## Features

- **14 Skills** - Productivity commands for common development tasks
- **11 AI Agents** - Specialized agents for different development roles

## Skills

| Skill | Description |
|-------|-------------|
| `/feature-plan` | Plan feature implementation with technical specs |
| `/new-task` | Analyze task complexity and create implementation plan |
| `/code-explain` | Generate detailed code explanations with diagrams |
| `/code-optimize` | Optimize code for performance and efficiency |
| `/code-cleanup` | Refactor and clean up code |
| `/lint` | Run linting and fix code quality issues |
| `/docs-generate` | Generate documentation |
| `/api-new` | Create new Next.js API route |
| `/api-test` | Generate API tests |
| `/api-protect` | Add security to API endpoints |
| `/component-new` | Create new React component |
| `/page-new` | Create new Next.js page |
| `/types-gen` | Generate Supabase TypeScript types |
| `/edge-function-new` | Create Supabase Edge Function |

## Agents

- **tech-stack-researcher** - Research technology choices
- **backend-architect** - Design backend systems
- **deep-research-agent** - Comprehensive research
- **frontend-architect** - Create accessible UIs
- **learning-guide** - Teach programming concepts
- **performance-engineer** - Optimize system performance
- **refactoring-expert** - Improve code quality
- **requirements-analyst** - Transform ideas into specs
- **security-engineer** - Identify vulnerabilities
- **system-architect** - Design scalable architecture
- **technical-writer** - Create documentation

## Installation

```bash
# Add the marketplace
/plugin marketplace add gpaura/claude_plugins

# Install the plugin
/plugin install gpaura-claude-code@gpaura-claude-code

# Enable the plugin
/plugin enable gpaura-claude-code@gpaura-claude-code
```
